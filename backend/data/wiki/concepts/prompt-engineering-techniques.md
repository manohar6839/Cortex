# Prompt Engineering Techniques

## Summary

Systematic approaches to crafting effective prompts including chain-of-thought, few-shot, and role-playing strategies.

Prompt engineering is the practice of designing inputs that elicit desired outputs from LLMs.

## Chain-of-Thought

Adding 'Let's think step by step' dramatically improves reasoning accuracy on math and logic problems.

## Few-Shot Learning

Providing 3-5 examples in the prompt helps the model understand the expected format and style.

## System Prompts

Role-based system prompts establish persona, constraints, and output format.

## Structured Output

Using JSON schemas and XML tags ensures parseable, consistent responses from LLMs.
