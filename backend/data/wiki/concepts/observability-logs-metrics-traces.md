# Observability: Logs, Metrics, Traces

## Summary

The three pillars of observability and how to implement them using OpenTelemetry, Prometheus, and Grafana.

Observability enables understanding system behavior from external outputs.

## Structured Logging

JSON-structured logs with correlation IDs enable efficient search and aggregation.

## Metrics

Prometheus time-series metrics with histograms, counters, and gauges for SLI/SLO monitoring.

## Distributed Tracing

OpenTelemetry traces follow requests across microservice boundaries with span contexts.

## Dashboards

Grafana unifies logs, metrics, and traces in actionable dashboards with alerting.
