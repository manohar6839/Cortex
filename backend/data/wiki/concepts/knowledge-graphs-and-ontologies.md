# Knowledge Graphs and Ontologies

## Summary

Knowledge graphs represent structured information as entities and relationships, enabling reasoning and inference.

Knowledge graphs store facts as (subject, predicate, object) triples.

## RDF and SPARQL

RDF (Resource Description Framework) provides a standard model for data interchange. SPARQL is the query language for RDF.

## Ontology Design

Ontologies define concepts, properties, and relationships within a domain, providing shared vocabulary.

## Graph Embeddings

TransE, RotatE, and ComplEx learn vector representations of entities and relations for link prediction.

## Applications

Knowledge graphs power search engines, recommendation systems, and biomedical research.
