# WebAssembly for Server-Side Computing

## Summary

WebAssembly is expanding beyond browsers into server-side and edge computing with WASI and component model.

WebAssembly (Wasm) provides a portable, sandboxed execution environment.

## WASI

The WebAssembly System Interface provides standardized access to OS resources like files and networking.

## Component Model

The component model enables composing Wasm modules with well-defined interfaces across languages.

## Edge Computing

Cloudflare Workers and Fastly Compute run Wasm at the edge with sub-millisecond cold starts.

## Language Support

Rust, C/C++, Go, and Python can compile to Wasm, with varying levels of maturity.
