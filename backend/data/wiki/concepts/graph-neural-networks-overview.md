# Graph Neural Networks Overview

## Summary

GNNs extend deep learning to graph-structured data using message passing, attention, and pooling mechanisms.

Graph Neural Networks learn representations over nodes, edges, and graphs.

## Message Passing

GCN and GraphSAGE aggregate neighbor features through iterative message passing layers.

## Graph Attention

GAT uses attention weights to learn importance of different neighbors dynamically.

## Graph Transformers

Recent architectures apply transformer attention to fully-connected graphs with positional encodings.

## Applications

Drug discovery, social network analysis, traffic prediction, and recommendation systems.
