# Zero-Knowledge Proofs Explained

## Summary

Zero-knowledge proofs allow one party to prove knowledge of a value without revealing the value itself.

ZKPs are a cryptographic method for verifiable private computation.

## zk-SNARKs

Succinct Non-interactive ARguments of Knowledge provide constant-size proofs but require a trusted setup.

## zk-STARKs

Scalable Transparent ARguments of Knowledge eliminate trusted setup using hash functions.

## Applications

Layer-2 rollups (zkSync, StarkNet) use ZKPs for Ethereum scaling with validity proofs.

## Circom and Halo2

Circuit languages like Circom and frameworks like Halo2 enable developers to write ZK programs.
