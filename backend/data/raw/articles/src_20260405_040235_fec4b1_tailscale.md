A secure, zero-trust mesh network connecting my Mac (development machine), Hetzner CX32 server (production infrastructure), and iPhone (mobile access) into a single private LAN using WireGuard-encrypted tunnels. Eliminates the need to expose ports publicly for SSH, dev servers, AI inference endpoints, and internal dashboards. Enables three core capabilities: hardened server access without public SSH exposure, instant private HTTPS access to any localhost service from any device, and a personal VPN through the Hetzner exit node in Helsinki. Foundation layer for future setups including private on-device AI inference (Gemma 4 / Ollama on Mac → phone), secure access to OpenClaw/NemoClaw agent infrastructure, and zero-config dev previews on mobile. All traffic stays device-to-device — Tailscale only coordinates key exchange, never touches actual data.
