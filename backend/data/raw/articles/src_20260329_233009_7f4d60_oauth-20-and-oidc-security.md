OAuth 2.0 is an authorization framework for delegated access to APIs.

## Authorization Code Flow

The most secure flow for web apps, using a server-side code exchange to obtain tokens.

## PKCE

Proof Key for Code Exchange protects public clients (mobile, SPA) from code interception attacks.

## OpenID Connect

OIDC adds an identity layer on top of OAuth 2.0, providing ID tokens with user claims.

## Token Management

Access tokens, refresh tokens, and token rotation strategies for secure session management.
